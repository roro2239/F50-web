短信转发

 
  • 当收到短信时，系统会通过预设的 shell 脚本来转发短信。在脚本中可以使用以下变量来动态获取短信的相关信息：
    • ${sms_content}：表示短信的内容。
    • ${sms_sender}：表示发件人的号码。
    • ${sms_time}：表示短信的接收时间。
    • ${sms_content_query_escape}：表示短信内容经过 URL 查询编码（URL.QueryEscape）后的结果，适用于需要将短信内容作为 URL 参数的场景。
    • ${sms_sender_query_escape}：表示发件人号码经过 URL 查询编码后的结果。
    • ${sms_time_query_escape}：表示接收时间经过 URL 查询编码后的结果。
  • 示例：
    • curl http://example.com/api?content=${sms_content_query_escape}&sender=${sms_sender_query_escape}&time=${sms_time_query_escape}
